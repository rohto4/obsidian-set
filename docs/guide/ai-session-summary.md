# AI Session Summary Guide

作成日: 2026-04-14
更新日: 2026-04-19

## 目的

Codex / Claude Code / Gemini CLI とのやり取りを、後から探索できる形で保存する。

## 保存先

- ユーザプロンプト全文とAI回答要約: `G:\knowledge-vault\prompts\`
- タスク実施概要と引き継ぎ: `G:\knowledge-vault\tasks\`
- LLM想起用短縮記憶: `G:\knowledge-vault\memory\`

## 保存するもの

- ユーザプロンプト全文
- AI出力の要約
- 実行した主な操作
- 参照した主なファイル / URL
- 判断待ち
- 次アクション

## 保存しないもの

- API key、token、credential
- 長いAI出力全文
- 内部推論ログ
- 著作権上保持すべきでない長文引用

## Template

正本:

```text
G:\knowledge-vault\templates\ai-session-summary.md
G:\knowledge-vault\templates\task-summary.md
```
